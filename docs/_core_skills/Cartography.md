__Time Required:__ Varies

__Description:__ The _cartography_ skill allows characters to read and create maps of various types, from topographical to political.

Action                             | Difficulty
---------------------------------- | ----------
Read maps and identify symbols     | Very Easy
Create small-scale maps            | Easy
Create terrain maps                | Moderate
Create world maps                  | Difficult
Create dimensional/space maps      | Very Difficult
